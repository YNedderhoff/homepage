  • Chu-Liu/Edmonds’ Dependency Parser, 2015

    Ein in Python implementierter Dependency Parser auf Basis des MST-Algorithmus von Chu-Liu und Edmonds. Die Software wurde ursprünglich im Rahmen des Dependency Parsing Kurses an der Universität Stuttgart entwickelt.

    Verfügbar unter der MIT-Lizenz.


  • Simple Tree Automaton, 2015

    Ein einfacher, in Python implementierter Baumautomat, der Anhand einer Grammatik und einer Liste von Startsymbolen für einen Baum entscheiden kann, ob er von der Grammatik akzeptiert wird, oder nicht. Die Software wurde ursprünglich im Rahmen des Tree Automata Kurses an der Universität Stuttgart entwickelt.

    Verfügbar unter der MIT-Lizenz.
